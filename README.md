# AI-Powered Test Automation Framework

## Overview
This project is a scalable test automation framework built using Selenium WebDriver, TestNG, and Maven. It integrates AI-assisted test generation using Python to convert user stories into test templates.

## Features
- Page Object Model (POM) design pattern
- 25+ automated UI test scenarios
- Covers Login, Products, Cart, and Checkout workflows
- ExtentReports integration for reporting
- Screenshot capture on failure
- TestNG-based execution with parallel support
- Maven build management

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Python (AI-based test generation)
- Jenkins (CI/CD ready)

## How to Run
```bash
mvn clean test
