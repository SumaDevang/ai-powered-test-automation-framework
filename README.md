# AI-Powered Test Automation Framework

## 🚀 Overview
This project is a scalable test automation framework built using Selenium WebDriver, TestNG, and Maven. It integrates AI-assisted test generation using Python to convert plain-English user stories into executable test templates.

The framework follows industry best practices and supports end-to-end automation for modern web applications.

---

## ✨ Key Features
- Page Object Model (POM) design pattern for maintainability
- 25+ automated UI test scenarios
- Covers Login, Products, Cart, and Checkout workflows
- ExtentReports integration for detailed reporting
- Screenshot capture on test failure
- Parallel test execution using TestNG
- Maven-based build and dependency management
- AI-assisted test generation using Python

---

## 🛠️ Tech Stack
- **Language:** Java, Python  
- **Automation:** Selenium WebDriver  
- **Testing Framework:** TestNG  
- **Build Tool:** Maven  
- **Reporting:** ExtentReports  
- **CI/CD:** Jenkins (ready for integration)
